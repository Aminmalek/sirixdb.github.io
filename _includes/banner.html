SirixDB

SirixDB is a hybrid disk-oriented and in-memory database system.
It has a lightweight low-overhead buffer-manager, which is able to store in-memory page references.
It is a log-structured temporal document store
and stores the histories of all resources in a space-efficient manner.
It supports the efficient reconstruction of any revision as well as comparisons.
A query processor supports sophisticated time travel queries.
See the Open Source repository.
